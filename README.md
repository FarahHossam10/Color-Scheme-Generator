# 🎨 Color Scheme Generator

A responsive web application that generates beautiful color palettes from a selected base color using **The Color API**. Users can choose from multiple color harmony modes and copy any generated hex color code with a single click.

## Features

* 🎨 Pick any base color using the native color picker
* 🌈 Generate color palettes in multiple harmony modes:

  * Monochrome
  * Monochrome Dark
  * Monochrome Light
  * Analogic
  * Complement
  * Analogic Complement
  * Triad
  * Quad
* 📋 Click any generated color or its hex code to copy the hex value to your clipboard
* 📱 Fully responsive layout for desktop and mobile devices

## Built With

* HTML5
* CSS3
* JavaScript (ES6)
* [The Color API](https://www.thecolorapi.com/)

## How It Works

1. Select a base color using the color picker.
2. Choose a color harmony mode.
3. Click **Get color scheme**.
4. Browse the generated palette.
5. Click any color to instantly copy its hex code.
